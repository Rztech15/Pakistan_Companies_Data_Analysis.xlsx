# Pakistan Top Companies Data Analysis Dashboard

## Project Overview
This project analyzes the revenue and market capitalization of top Pakistani companies using Microsoft Excel. 
The goal of the project is to present financial comparisons and insights through a clean and interactive dashboard.

## Tools Used
- Microsoft Excel
- Data Visualization
- Dashboard Design

## Dataset
The dataset includes financial information of the following companies:
- Pakistan State Oil (PSO)
- Oil & Gas Development Company Limited (OGDCL)
- Habib Bank Limited (HBL)
- Lucky Cement
- United Bank Limited (UBL)
- Meezan Bank

Data includes:
- Revenue (PKR Billion)
- Market Capitalization (USD Billion)

## Dashboard Features
The Excel dashboard includes:

- Key Performance Indicators (KPI)
- Revenue comparison bar chart
- Market capitalization comparison chart
- Revenue share pie chart
- Insights and analysis

## Key Insights
- Pakistan State Oil (PSO) has the highest revenue among the listed companies.
- United Bank Limited (UBL) has the highest market capitalization.
- The energy sector contributes a large share of total company revenue.
- Banking companies show strong market valuation.
